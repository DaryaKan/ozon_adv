# ozon_adv

Статический прототип (HTML/CSS/JS). После правок задеплойте актуальные файлы на GitHub Pages (репозиторий сайта, например `username.github.io`, или ветка `gh-pages` / папка `docs` в настройках Pages).

Внизу страницы создания объявления в футере указана версия (`prototype-build`). Если на сайте не та версия, что в репозитории — сделайте push в репозиторий Pages, подождите 1–2 минуты и откройте страницу с параметром `?_p=...` из `create-ad.html` или обновите с принудительным сбросом кэша.
